## Placeholder Guide

- `{research_topic}`: The primary topic researched
- `{timestamp}`: Completion timestamp in YYYYMMDD-HHmm format
- `{research_focus}`: Specific research focus or angle
- `{target_audience}`: Intended audience for the report
- `{chapter_count}`: Number of chapters completed
- `{source_count}`: Total number of sources with URLs
- `{chapter_N_title}`: Title for each chapter
- `{list_*}`: Bulleted or numbered lists of items
- `{describe_*}`: Detailed descriptive paragraphs
- `{executive_summary_paragraph_*}`: High-level summary paragraphs

## Notes

- All sources must use format: `[Source Title](full-url)` - NO exceptions
- Executive summary should be 2-3 paragraphs maximum
- Each chapter should have clear sections: Overview, Key Points, Analysis, Sources
- Mark information currency: (Web 2025), (Training Data), (Verified 2025)
- Sources section should distinguish between current and training data
- Report should be professionally formatted and publication-ready
- All web sources must be accessible and include search dates
