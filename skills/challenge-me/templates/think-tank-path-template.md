- `{cycle_count}`: Total number of cycles completed
- `{initial_thoughts}`: User's starting ideas
- `{codebase_path_or_none}`: Path to codebase or "None"
- `{documentation_path_or_none}`: Path to documentation or "None"
- `{web_urls_or_general}`: Specific URLs or "General search"

- `{list_*}`: Bulleted or numbered lists
- `{describe_*}`: Descriptive paragraphs
- `{summarize_*}`: Summary paragraphs
- `{repeat_for_*}`: Repeat the block for additional items
