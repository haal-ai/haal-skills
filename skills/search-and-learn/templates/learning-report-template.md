## Placeholder Guide

- `{topic}`: The learning topic or question being researched
- `{YYYYMMDD-HHmm}`: Timestamp in OLAF standard format
- `{specific_question_or_skill}`: The specific learning objective
- `{knowledge_level_description}`: Current knowledge level and experience
- `{application_context}`: Related skills, time constraints, application context
- `{Concept_N}`: Name of a key concept discovered
- `{description}`: Free-form descriptive text
- `{key_point_N}`: Specific point about a concept
- `{Insight_N}`: Main insight or takeaway
- `{explanation}`: Explanation of an insight
- `{connection_N}`: Connection to existing knowledge
- `{Source_Title_N}`: Title of a source
- `{url_N}`: Full URL to source (MANDATORY)
- `{type}`: Source type (Documentation, Academic, Blog, Forum, Video, Tutorial)
- `{what_it_provided}`: Key contribution from the source
- `{count}`: Numeric count
- `{use_case_title}`: Title of a practical use case
- `{when_this_applies}`: Context for when to use this
- `{language}`: Programming language for code blocks
- `{practical_example}`: Code or configuration example
- `{what_this_achieves}`: Expected outcome
- `{Gap_Area_N}`: Area requiring additional research
- `{relevance}`: Why this gap is important
- `{where_to_look}`: Suggested resources for filling gap
- `{High|Medium|Low}`: Priority level
- `{Action_N}`: Specific action to take
- `{what_to_achieve}`: Objective of an action
- `{what_to_use}`: Resources needed
- `{duration}`: Time estimate
- `{Learning_Goal_N}`: Short-term learning goal
- `{how_to_learn}`: Approach for learning
- `{how_to_measure}`: Success criteria
- `{Development_Goal_N}`: Long-term development goal
- `{topic_N}`: Follow-up topic
- `{why_relevant}`: Relevance of follow-up topic

## Notes

- **MANDATORY URL Collection**: Every source MUST include full URL - NO generic references allowed
- **Source Format**: All sources must use format: `[Source Title](full-url)`
- **Quality Assessment**: Rate sources on 5-star scale based on authority, accuracy, and relevance
- **Practical Focus**: Always include concrete examples and applications
- **Gap Identification**: Be honest about what wasn't fully understood or needs more research
- **Validation**: Include understanding checks and practical exercises
- **Connections**: Link new knowledge to existing knowledge for better retention
- **Next Steps**: Provide clear, actionable recommendations for continued learning
