# Augment Code Unit Test: Step-by-Step Tutorial

How to Execute the "Augment Code Unit Test" Workflow

This tutorial shows how to use the Developer competency's unit test enhancement workflow to improve test coverage and quality.

## Prerequisites

- OLAF Framework with developer competency pack
- Existing codebase with unit tests
- Testing framework in place

## Step-by-Step Instructions

### Step 1: Initialize Test Augmentation
**User Action:** Execute `olaf augment tests` or `improve test coverage`

### Step 2: Specify Code and Test Parameters
**Provide:** Code location, test framework, coverage goals

### Step 3: Test Analysis and Enhancement
**System:** Analyzes existing tests, identifies gaps, generates new tests using Propose-Act protocol

## Expected Timeline: 15-30 minutes
