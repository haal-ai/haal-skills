# Assess GenAI Initiative Idea

## Overview

Evaluates GenAI initiative ideas through research-driven analysis and challenge-based questioning to determine feasibility, value, and implementation approach.

## Purpose

Decision-makers often need to evaluate whether a GenAI initiative is worth pursuing. This competency provides structured analysis combining market research, technical feasibility assessment, and critical challenge questions to make informed go/no-go decisions.

## Usage

**Command**: `assess genai initiative`

**Protocol**: Propose-Act

**When to Use**: When considering a new GenAI project, feature, or initiative and need objective evaluation before committing resources.

## Parameters

### Required Inputs
- **Initiative Description**: Brief description of the GenAI idea or initiative being considered

### Optional Inputs
- **Business Context**: Industry, company size, existing tech stack
- **Constraints**: Budget, timeline, team capabilities

### Context Requirements
- Clear understanding of the initiative's intended purpose
- Access to relevant business context

## Output

**Deliverables**:
- Research summary on similar initiatives
- Feasibility assessment
- Challenge questions and considerations
- Recommendation (proceed/modify/abandon)

**Format**: Structured markdown report with sections for research staging, analysis, and recommendations

## Examples

### Example 1: Customer Service Chatbot

**Scenario**: Company considering AI chatbot for customer support

**Command**:
```
assess genai initiative
```

**Input**: "We want to build an AI chatbot to handle tier-1 customer support inquiries"

**Result**: Analysis covering existing solutions, implementation complexity, ROI considerations, and specific challenges for the use case

### Example 2: Document Analysis System

**Scenario**: Legal firm evaluating document review automation

**Command**:
```
assess genai initiative
```

**Result**: Research on legal AI tools, compliance considerations, accuracy requirements, and implementation roadmap

## Related Competencies

- **should-i-use-ai**: Use this for simpler yes/no AI appropriateness decisions

- **use-skill**: Can route to this skill when initiative assessment is needed

## Tips & Best Practices

- Provide specific details about your use case for more targeted analysis
- Be honest about constraints - better to know early if something won't work
- Use the challenge questions to stress-test your assumptions
- Consider starting with a pilot/MVP based on recommendations

## Limitations

- Analysis quality depends on publicly available information
- Cannot access proprietary competitive intelligence
- Recommendations are guidance, not guarantees of success
