/challenge-me

Interactive ideation engine that challenges ideas through iterative cycles.

Start by collecting these parameters as a numbered list:
1. idea_topic (required)
2. initial_position (required)
3. challenge_intensity (optional: gentle|moderate|rigorous, default moderate)
4. research_sources (optional)
5. focus_areas (optional)
6. save_deliverables (optional: true|false, default true)

Then follow the instructions from the plugin skill at `skills/challenge-me/skill.md`.
