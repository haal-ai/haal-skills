# Tutorial: Select Competency Collection

## Overview
Choose and activate specific OLAF competency collections based on your persona or team needs.

## Prerequisites
- OLAF framework installed
- Python environment configured
- Understanding of available collections

## Estimated Time
10-15 minutes

## Steps

### Step 1: Invoke Select Collection
```
olaf select collection
```

### Step 2: Choose Collection Name
Select from available collections or specify "custom".

### Step 3: Specify Custom Competencies (if applicable)
If using custom collection, list specific competencies to include.

### Step 4: Confirm Index Update
Approve regeneration of competency index.

### Step 5: Wait for Reindexing
Allow 1-2 minutes for framework reindexing to complete.

### Step 6: Verify Collection Activation
Test that relevant competencies are now available.

## Verification Checklist
- [ ] Collection activated
- [ ] Competency index regenerated
- [ ] Condensed framework updated
- [ ] Relevant competencies accessible
- [ ] Framework performance improved

## Troubleshooting
- **Script not found**: Verify Python environment and script path
- **Reindexing failed**: Check for framework corruption
