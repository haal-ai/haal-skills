How to Execute the "Generate Tech Spec From Code" Workflow

Step 1: Initialize Spec Generation¶

User Action: Execute olaf tech spec from code or extract spec

Step 2: Provide Code Parameters¶

Specify: Code location, output format, specification depth

Step 3: Code Analysis and Spec Generation¶

System: Analyzes code structure, extracts patterns, generates technical documentation using Act protocol
