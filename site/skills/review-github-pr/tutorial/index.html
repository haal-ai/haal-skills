Review GitHub PR: Step-by-Step Tutorial¶

How to Execute the "Review GitHub PR" Workflow

Comprehensive pull request review with integration analysis and action planning.

Prerequisites¶

  • OLAF Framework with developer competency pack
  • GitHub repository access
  • Pull request to review

Step-by-Step Instructions¶

Step 1: Initialize PR Review¶

User Action: Execute olaf review pr or github pr

Step 2: Specify PR Details¶

Provide: Repository, PR number, review focus areas

Step 3: PR Analysis and Review¶

System: Reviews changes, analyzes integration impact, generates action plan using Propose-Act protocol

Expected Timeline: 15-25 minutes¶
