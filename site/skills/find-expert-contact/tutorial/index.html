Find Expert Contact: Step-by-Step Tutorial¶

How to Execute the "Find Expert Contact" Workflow

This tutorial shows how to locate subject matter experts within your organization.

Prerequisites¶

  • Clear understanding of the expertise you need
  • Access to organizational directory or contact information
  • Context about why you need the expert (optional but helpful)

Step-by-Step Instructions¶

Step 1: Invoke the Competency¶

Initiate the expert search

User Action: 1. Type the command: find expert contact 2. Describe the expertise area you need

OLAF Response: OLAF will acknowledge the request and begin searching for relevant experts

Step 2: Provide Expertise Details¶

User Action: Describe what expertise you need

I need someone who can help with PostgreSQL database performance optimization
and query tuning for a high-traffic application

Provide Context (Optional): - Problem: Slow query performance affecting user experience - Team: Backend engineering - Urgency: Need help this week

Step 3: Expert Search¶

What OLAF Does: - Searches organizational directory - Identifies people with matching expertise - Reviews roles, teams, and backgrounds - Ranks by relevance and availability

You Should See: Search in progress confirmation

Step 4: Review Expert Contacts¶

What OLAF Provides: - Primary expert contact with details - Alternative contacts if available - Expertise areas and background - Best contact method or availability notes

You Should See: Structured list of expert contacts with relevant information

Step 5: Follow Up¶

User Action: Use the contact information - Reach out to the identified expert - Mention the specific expertise area - Provide context about your need - Ask follow-up questions if needed

Verification Checklist¶

✅ Expert contact information is complete (name, email, role) ✅ Expertise areas match your needs ✅ Alternative contacts provided if available ✅ Contact method or availability notes included

Troubleshooting¶

If no experts found: - Broaden the expertise area description - Try related or adjacent expertise areas - Ask for external resources or documentation

If multiple experts returned: - Ask OLAF to recommend the best match for your specific situation - Request ranking by availability or expertise depth

If contact information seems outdated: - Verify through organizational directory - Ask for alternative contacts - Request most recent information available

Key Learning Points¶

  1. Specific Descriptions: More specific expertise descriptions yield better matches
  2. Context Matters: Providing context helps identify the right expert
  3. Alternatives: Always good to have backup contacts
  4. Organizational Knowledge: Leverages org structure and role information

Next Steps to Try¶

  • Contact the identified expert
  • If implementing an initiative, use assess genai initiative first
  • Save expert contacts for future reference
  • Build your network of go-to experts

Expected Timeline¶

  • Total search time: 2-3 minutes
  • User input required: Expertise description (1 minute)
  • OLAF execution time: Search and ranking (1-2 minutes)
